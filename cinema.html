Cinema - 'App'

2 December, 2015

Final Design: Displayed on a mockup of an iPhone

OVERVIEW

A UI/UX design for a Cinema application. The goals of this project were to create a simple and clear interface that gives the user a relevant amount of features. The navigation is designed to make purchasing tickets for the user an easy experience with options such as choosing a venue or film.

Features

Compatible with Smartphones

User-Friendly Navigation

OUTCOME

Final Designs

The outcome of this project includes four different screen design each showing the funtionality and user interface of the design. The screen designs were produced for a concept of a cinema application that could be used on smartphones.
