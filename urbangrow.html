Urban Grow

3 November, 2016

A work in progress project, view here! (desktop only)

OVERVIEW

A website that can be used for tutorials on growing miniature plants such as Cacti, Bonsai and Pumpkins. The style of design is similar to that of a blog. My goal with this project was to experiment in web design and continue to develop my own style. The outcome is ongoing and the goal is to develop a fully responsive website.

Features

Fully Responsive

User-Friendly Navigation

DEVELOPMENT

Introduction

The current version of this project is in the basic design and development stages featuring some simple pages and navigation. To continue the project I am going to be adding content gradually for the purpose of the website.
