Origami - 'Web App'

27 April, 2016

Live Project for an Origami Web Application, view here! (for use with smartphones only)

OVERVIEW

A web application that teaches users to create various types of Origami such as planes, boats and animals. The purpose of this project was to challenge my self in my third and final year of University. The development process required me to explore UI/UX Design, Illustration and coding. There is also a live version of the project that it is currently only compatible with smartphones.

Features

Compatible with Smartphones

User-Friendly Navigation

DEVELOPMENT

Introduction

Over twelve weeks my challenge was to design and develop a fully functional web app that could be used with smartphones. The initial idea for the app was based on tutorials for making paper planes. This later expanded to various types of Origami.

The first stage of the design process was to create the content. This included artwork and text that would be featured within the application. Creating the artwork consisted of some draft sketches drawn from observation that were later illustrated and optimized for web.

Basic Sketches: A variety of artwork features that show the origami designs

Concept Designs: Illustrated and vetorized artwork for the in app designs

OUTCOME

Final Designs

The project outcome is a live web application tha can be used from the link above the page. The overall design includes a variety of screen and user interface funtionalities. These also include artwork with instructions of creating Origami within the app.
