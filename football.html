Football Form Labs

22 November, 2016

Final Design: Displayed on a mockup of an iPhone

OVERVIEW

A recent User Interface project for a company called Football Form Labs. The challenge was to create a design that would be suitable for all users including those specifically interested in football betting. The outcome of the design is elegant and focuses on creating a user experience that relates to football.

Features

User-Freindly Navigation

Compatible with Smartphones

Engaging Visuals

DEVELOPMENT

Introduction

After attending an interview with Football Form Labs as a graphic designer, the company asked if I could produce a small project for them to help with the applicant decisions process. This project began with a basic wireframe of a widget that they had asked to be re-designed. One feature they were particularly asking for is the design to be extremeley clear and understandable to all users.

The purpose of this application is to provide people interested in the betting industry with football based statistics. For example, this may include football match statistics that show the teams that are most likely to win.

Sketches

With the project brief in mind the first step began with some basic sketches of user interface wire-frames. Some of my initial design ideas were to use simple drop down menus for each section featured within the widget/app.

The interface design had to be user-friendly and easyily understandable for the users. This meant drawing up concepts and using some of the most modern design trends including menu icons/buttons for good functionality.

Sketches: Original sketches by my self illustrated and displayed on a mockup sketchbook

OUTCOME

Final Designs

The design outcome is a variety of user interfaces displaying each screen of the widget or app. These images were created in Photoshop and produced in a variety of formats optimized for smartphones and as a .psd file. The overall project features 5 different screens that showcase the basic funtionality that would be seen within the application.
